## Electricity Consumption Analysis

This project analyzes electricity consumption data in France, focusing on different customer categories: `Entreprises`, `Professionnels`, and `RÃ©sidentiels`. The analysis involves:

- **Data Cleaning**: Handling missing values and ensuring the dataset is ready for analysis.
- **Visualization**: Visualizing trends in electricity consumption over time and across different customer categories.
- **Clustering**: Using K-Means clustering to identify patterns in electricity consumption.

### Files in the Repository
- **Taxable.ipynb**: Jupyter Notebook for performing data analysis, visualization, and clustering.
- **cons_elec_france.csv**: The dataset containing electricity consumption data for France.

### How to Run the Project
1. Clone the repository.
2. Install the necessary libraries:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the `Taxable.ipynb` notebook in your Jupyter environment.
